CRUD
C - insert
R - select
U - update 
D - delete

R
select 컬럼명 from 테이블명 where 컬럼명 = '컬럼내용';

C
insert into 테이블명 (컬럼명) values (컬럼내용);

U
update 테이블 set 컬러명=컬럼내용 where 대상컬럼명=컬럼내용;

D
delete 테이블 form where 컬럼='컬럼내용';
